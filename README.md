# To-Do List (Python CLI)

A simple console-based to-do list manager built in Python.  
It allows users to:
- Add tasks
- View pending, completed, and deleted tasks
- Mark tasks as finished
- Delete tasks
- Exit gracefully with error handling

---

## Features
- Menu-driven interface
- Separate lists for pending, completed, and deleted tasks
- Input validation with `try/except`
- Clear formatted output
